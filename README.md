# Tabby Self-Hosted on Render

This repository runs Tabby (open-source AI coding assistant) on Render using Docker.

## Deployment
1. Fork or clone this repo
2. Create a Render Web Service
3. Select Docker runtime
4. Deploy

## Default Port
8080

## Usage
Connect your VS Code or JetBrains Tabby plugin to:
https://YOUR-RENDER-URL
